* How to Install via Makefile.PL

perl Makefile.PL
make 
make test
make install

* How to Build From Scratch

Download UMMF source archive from http://sourceforge.net/projects/ummf/
cd ummf-*
make 

* How to find out more

Point your HTML browser at http://ummf.sourceforge.net/ or ./doc/html/index.html
